HackerYou Web Accessibility Club

A group of curious junior developers working to make the web an inclusive space.

About the Club

The HackerYou Web Accessibility Club was founded by a group of junior web developers studying at HackerYou in Toronto, Ontario.

Our goal is to bring students together from diverse backgrounds and unite them under the common objective of understanding the basic principles of universal design and web accessibility.

View our Code of Conduct for more information on our inclusivity policies.

Welcome to the World of WCAG 2.0!

When: February, 2017

Where: To Be Determined

You don’t need to be a WCAG wizard to write accessible code. You don’t even need to know that WCAG stands for Web Content Accessibility Guidelines. Just bring your laptop and hang out with us for an hour!

In this session you will learn:

  • How to use automated tools and manual tests to evaluate a site’s accessibility level
  • How to read an accessibility evaluation with reference to WCAG
  • The difference between WCAG levels A - AAA
  • How to find solutions to common accessibility issues through WCAG

Meet Your Screenreader!

When: April, 2017

Where: To Be Determined

Testing your site for screen reader users can be difficult if you don't know how to use one. Bring your laptop (and your headphones) and we'll walk you through the ins and outs of using and testing sites with a screen reader.

In this session you will learn:

  • A hands-on session geared towards getting comfortable with a screenreader (VoiceOver for Mac userse or NVDA for Windows users)
  • How to navigate a website with a screen reading device
  • How to find and implement solutions to common issues for screen reader users

A11Y and JavaScript!

When: June, 2017

Where: To Be Determined

Accessibility doesn't mean you have to forego functionality. Join us for a code along where we show you how to solve common accessibility issues with JavaScript!

In this session you will learn:

  • How to build an accessible modal box
  • How to keep websites functional without sacrificing performance
  • How to create accessible tooltips
  • How to build an accessible slider/carousel
